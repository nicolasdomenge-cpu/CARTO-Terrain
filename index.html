Configuration des champs

Cette exportation contient uniquement la structure des champs : noms, types, listes suggérées et valeurs proposées. Elle ne contient pas les points saisis.

Pour les champs Texte, Entier ou Décimal, tu peux activer une liste suggérée. Les valeurs doivent être séparées par un point-virgule ;. Exemple : chêne ; hêtre ; pin ou 1 ; 2 ; 3 ; 4.
Tu peux aussi choisir un seul champ comme étiquette pour l’afficher directement sur la carte au-dessus des points.
